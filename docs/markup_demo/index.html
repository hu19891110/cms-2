    General Page Layout

    Column Layout

    Overall site width is determined by the width on the #wrapper <div>

    Page layout is determined by the class applied to the #wrapper <div>

    Default values are .one-column, .two-column, .two-column-alt and .three-column

    Column and gutter widths are configurable inside page.css

    .one-column #nav-col  { display: none; }
.one-column #main-col { width: inherit; float: none; padding: 0; }
.one-column #sub-col  { display: none; }

.two-column #nav-col  { display: none; }
.two-column #main-col { width: 590px; float: left; padding: 0; }
.two-column #sub-col  { width: 270px; float: left; }

.two-column-alt #nav-col  { width: 220px; float: left; }
.two-column-alt #main-col { width: 640px; float: left; }
.two-column-alt #sub-col  { display: none; }

.three-column #nav-col  { width: 180px; float: left; }
.three-column #main-col { width: 420px; float: left; }
.three-column #sub-col  { width: 220px; float: left; }

    Forms

    General form guidelines

    • wrap fields in <div> with the class of .element
    • if extra spacing is needed below the field item, replace the <div> with <p>

    Field label & single input field

    <div class="element">            
    <label for="fname">First Name</label>
    <input id="fname" type="text" class="field" name="" value="Input field" />
</div>

    Use this markup for text and password input types

    Include the identifier .field

    Field label & single textarea

    <div class="element">            
    <label for="comment">Comment</label>
    <textarea id="comment" class="md">Text area text</textarea>
</div>

    Classes .sm, .md, .lg and .xlg can be added to modify the height of the field

    Field label and select menu

    <div class="element">
    <label for="options">field label &amp; select menu</label>
    <select id="options">
        <option>Option 1</option>
        <option>Option 2</option>
        <option>Option 3</option>
    </select>
</div>

    Field with a note or hint

    <div class="element">            
    <label for="comment">Comment</label>
    <span class="note">Please limit to 200 characters</span>
    <textarea id="comment" class="md">Text area text</textarea>
</div>

    Add a <span> with a class of .note

    Can be inserted before or after field, or inside a <label>

    Please limit to 200 characters

    Buttons

    Default button styles will be applied to:

    <button>Submit</button>
    or 
    <input class="button" type="submit" />Submit<button>
    or 
    <a class="button" href="/buy">Buy Now</a>

    Buttons with contextual link or hint

    It may be useful to provide text next to buttons that aligns with the button labels, in this case use a span inside the .buttons container

    <div class="buttons">
    <a class="button" href="/buy">Take Tour</a>
    <a class="button" href="/buy">Start Trial</a>
    <a class="button" href="/buy">Buy Now</a>
    <span>This is a contextual clue with a <a href="">link</a></span>
</div>

    Notifications & Clues

    Notifications

    Used to provide the user with a general notification of some sort

    <div class="notification">There are no posts in this category.</div>

    Add a <div> with a class of .notification (in cases where bottom-margin is desired, this class may be used on a <p>)

    Can be modified with .positive, .negative, and .neutral classes to change colors from default (gray) to green, red, or yellow respectively

    Please enter your search terms.

    There was an error in your form.

    Your settings have been successfully changed.

    There are no posts in this category.

    Flag Notifications

    Used to provide a notification box with an arrow pointing to the relevant subject matter.

    <div class="flag flag-up negative">
    <span>You've left this field empty.</span>
</div>

    Add a <div> with a class of .flag, with a <span> wrapped around the content (in cases where bottom-margin is desired, the <div> may be replaced with a <p>)

    Use .flag-up, .flag-down, .flag-left, and .flag-right classes to specify the flag direction

    Can be modified with .positive, .negative, and .neutral classes to change colors from default (gray) to green, red, or yellow respectively

    You've left this field empty.

    This block has been successfully saved.

    Are you sure about this change?

    Contextual Clues

    Can be used to color a block of text.

    <p class="negative">Please enter an address</p>
    or 
    <p><strong class="positive">Maximum Security</strong></p>

    Use .positive, .negative, and .neutral classes to apply appropriate color

    You've chosen Maximum Security

    Growl-esque Notifications

    Growl notifications can be used to agrregate errors on forms and other user interactions.

    <ul id="growl">
    <li class="notification negative">>Your form is missing information, please 
    complete the required fields.</li>
</ul>

    Use .positive, .negative, and .neutral classes on the <li> to apply appropriate color
